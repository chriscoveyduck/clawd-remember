# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| latest  | ✅ |

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Email: chris@anthillmob.net

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fixes

You'll get a response within 48 hours. If confirmed, we'll aim to release a fix within 7 days and credit you in the changelog unless you prefer otherwise.
